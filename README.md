# 97Admin
97Admin后台管理系统是基于Codeigniter框架和Gentelella后台模版后台模版进行开发的。

### 服务器要求
* PHP >= 7.1.0
* OpenSSL
* mysqli
* Mbstring

### 安装说明：      
* 1.将97admin.sql文件导入到数据库中。
* 2.修改文件App/config/database.php。 
* 3.填写正确的数据库配置信息。 
* 使用测试帐号登录：admin  admin888
